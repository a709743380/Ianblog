Asynchronous JavaScript and XML

jQuery AJAX 常用參數說明

url 要請求的 URL。
type HTTP 方法，常見的有 GET、POST 等，默認為 GET。
method 與 type 等價，建議使用 method（因為與標準的 'fetch' 方法一致）。
data 傳遞到伺服器的資料，可以是物件或字串。
dataType 預期的返回數據類型，例如 'json'、'xml'、'html' 或 'text'。
contentType 請求的內容類型（默認為 'application/x-www-form-urlencoded; charset=UTF-8'）。
async 是否以非同步方式發送請求（默認為 'true'）。
cache 是否對請求的結果進行快取（默認為 true）。
headers 自訂 HTTP 標頭，例如：'{ 'Authorization': 'Bearer token' }'。
timeout 設定請求的超時時間（以毫秒為單位）。
beforeSend 發送請求之前執行的回調函數（function），接受一個 'jqXHR'(XML HttpRequest) 對象作為參數。
complete 無論成功或失敗，都會執行的回調函數（function）。
success 請求成功時執行的回調函數（function），接收回應的數據、文字狀態以及 `jqXHR` 對象作為參數。
error 請求失敗時執行的回調函數（function），接收 `jqXHR` 對象、文字狀態和錯誤訊息作為參數。
processData 是否將數據轉換為查詢字串（對於 'GET' 和 'POST' 默認為 'true'）。
crossDomain(Cross-Origin Requests) 是否允許跨域請求（通常自動判斷）。
xhrFields 用於設定 XMLHttpRequest 對象的屬性，例如'{ withCredentials: true }'。
